# 网页版协同决策记录

更新时间：2026-06-23

## 项目定位

本仓库用于开发个人本地看盘与量化研究软件，当前优先服务指数基金、行业 ETF、行业主题基金研究。

第一阶段目标：

```text
数据获取 -> 本地存储 -> 指标计算 -> 策略管理 -> 轻量回测 -> 报告展示 -> 桌面 UI
```

当前不接入真实自动下单，仅用于研究、学习和模拟。

## 当前限制

1. PySide6 桌面 UI 尚未开发。
2. AKShare 真实数据访问当前可能受网络代理影响。
3. 回测引擎当前仅支持单标的、日线、无杠杆。
4. 用户策略代码尚未做沙箱隔离。
5. 尚未实现 HTML/PDF 回测报告导出。
6. 尚未接入提醒系统、模拟账户和策略组合管理。
